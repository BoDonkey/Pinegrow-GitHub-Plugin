Branch existing Repo

This will create a branch of a repo already on GitHub and save a local copy. If you have a local project that you want to branch you must first create a new repo and then branch it.

You can optionally create a branch from an existing branch of a repo by entering the name. Leaving this blank will create a branch off of the Main branch.

In order to create a branch and new local copy of an existing repo you need to select a directory path and a folder name.

The folder will be created or overwritten, in the selected directory and the repo contents saved inside.
